# product-boxes-elementor

## Installation
- pull repository:<br/>
https://github.com/igorb0214/product-box-elementor.git (current repository)
- change volumes in `docker-compose.yml` file to local project path.<br/>


## Running the app
creat container:<br/>
```bash
docker-compose up
```
## Example
run `http://localhost` url in your browser.
